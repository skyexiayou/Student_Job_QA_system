# 使用说明

## 1. 登录与注册

打开系统后先注册账号。用户名至少 3 个字符，密码至少 6 个字符。注册成功后会自动登录。

## 2. 求职问答

登录后可直接提问，例如：

- 我没有实习经历，可以找什么工作？
- RAG 项目写进简历应该怎么描述？
- 后端开发面试应该重点复习什么？
- 校招 offer 对比应该看哪些因素？

系统会流式输出回答，并展示参考来源片段。

## 3. 知识库上传

右侧“上传知识文档”支持：

- PDF
- TXT
- Markdown

上传后系统会自动解析、清洗、分块并重建索引。

## 4. 建议扩充的知识库材料

建议优先收集以下资料，整理为 PDF、TXT 或 Markdown 后上传：

1. 学校就业网资料：校招通知、宣讲会信息、就业质量报告、签约流程说明。
2. 企业官方招聘资料：岗位 JD、校招 FAQ、岗位培养计划、招聘流程。
3. 岗位能力说明：后端、前端、测试、数据分析、产品、运营、算法等岗位能力模型。
4. 简历指导资料：优秀简历样例、项目经历写法、不同岗位简历模板、常见避坑。
5. 面试资料：HR 面题库、技术面题库、行为面 STAR 案例、面试复盘模板。
6. 笔试资料：编程题、行测题、专业基础题、数据分析 SQL 题。
7. 行业报告：互联网、金融、制造、国企、AI 应用等行业趋势与岗位变化。
8. 校友经验：真实求职复盘、面经、offer 对比经验、签约注意事项。
9. 政策文件：三方协议说明、劳动合同注意事项、档案户口相关说明。
10. 项目案例：课程设计、实习项目、竞赛项目如何包装为简历项目。

## 5. 会话操作

- “新建咨询”：开启新的对话上下文。
- “清空当前对话”：清除当前会话历史。
- “召回片段”：控制每次从知识库召回多少条参考内容。

## 6. 团队访问

如果要让同校园网伙伴访问，请按部署手册使用：

```bash
D:\Anaconda\python.exe run.py --host 0.0.0.0 --port 8000
```

同学访问：

```text
http://你的IPv4地址:8000
```
